Usage  notes
----------------------------------------------
- This program checks whether the riders can go on the rides at the amusement park
- First the user has to specify the minimum height for each ride in the amusement park then it will be checked with the height of the riders to decide whether the rider can go on the ride or not.
- Enter the information as follows: HEIGHT RIDE
- This program can be run using OMEGA server using g++ compiler
----------------------------------------------

Installation notes
----------------------------------------------
- This program does not require any installation.
